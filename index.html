This uses window.postMessage to send a message from this iframe to the parent page.

Clicking the button, or in our real-world case, submitting the form, triggers a secure message from the embedded iframe to the page on which it is embedded. The page then takes whatever action we tell it to (in this case, we change colour to celebrate and then re-direct to a thank-you page).

Here's the code we're using in the iframe:

document.getElementById("go").onclick = function() { window.parent.postMessage('hello from the iframe', 'https://www.unb.ca'); };
